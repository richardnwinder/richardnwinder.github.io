Git Commands

Use the links to explore each command

Config Branch File Local Repository Remote Repository

Local Configuration

Set global user name:

$ git config --global user.name “username”

Set global user email:

$ git config --global user.email “email-address”

Remote Repository

Push branch to remote repository:

$ git fetch origin
$ git push origin/branch-name

Local Repository

Clone remote repository:

$ git clone https://gitlab.com/repo/project.git

Overwrite local repository:

$ git fetch origin
$ git reset --hard origin/master

Overwrite local branch:

$ git fetch origin branch-name
$ git reset --hard origin/branch-name

Merge remote repository:

$ git pull origin

Merge branch from remote repository:

$ git pull origin/branch-name

Merge branch into master:

$ git checkout master
$ git merge branch-name

Local Branch

Create new branch:

$ git branch branch-name

Change working branch:

$ git checkout branch-name

Local File

Git current status:

$ git status

Git add file:

$ git add file-name

Git remove file:

$ git rm file-name

Git commit files:

$ git commit -m commit-message

Git diff:

$ git diff
