Go-Encrypt

Go-Encrypt is a user service connecting to the USB EnigmaKey encryption device. The service runs on localhost port 46502. The service api is callable from any webpages implementing the go-encrypt interface.

    EHT management

    Open the EHT Manager

  • Click on the EHT Manager link.

    • A list will be displayed showing the current EHT loaded into the EnigmaKey.

    Create a new EHT

  • Click the button [Create EHT].
  • Enter a new name for the EHT in the popup text box.
  • Click the Create button.
  • Cancel the action (click the button Create EHT again).

    • A new EHT will be created at the next available position in the EnigmaKey EHT table.

  • The EnigmaKey stores internally generated random enhanced hash tables (EHT), which can be selected to carry out encryption internally within the device, or to produce temporary hash tables, to be used with host encryption.

    • Installation Instructions.

    In order to run the Go-Encrypt webapp demo it is necessary to have obtained a USB EnigmaKey encryption device.


    Download the correct installer for your system, from the Downloads section.


    Windows Installer.

  • Unzip the windows package
  • Double Click or use the menu Open command to launch the installer - go-encrypt-0.1-windows-amd64.exe .
  • Click accept to run with administrator priviledges.
  • We would recommend installing with default options, so just click accept on all of the prompts.
  • The program should take just a few seconds to install.

    • Mac OSX Installer.

  • Open the disk image - go-encrypt-0.1-darwin-amd64.dmz -
  • Press and drag the Go-Encrypt.app into the Applications folder and release.
  • The Go-Encrypt apllication will now be installed in your Applicatios directory and is ready to use.

    • Linux Installer.

  • Copy the package tarball - go-encrypt-0.1-linux-amd64.tar.gz to a temporary directory. Example: 'home/user/test' directory.
  • Unpack the package tarball - go-encrypt-0.1-linux-amd64.tar.gz in the temp directory.
  • In the temp directory [Right Click] on the file - install.sh - and use the menu Open command to launch the installer.
  • Enter your user password to run with administrator priviledges.
  • When prompted type 'y' and press [enter] to create new encryption keys for the admin server.
  • The program should take just a few seconds to install.
