JS-Quick

JS-Quick came about as a need to have an easily accessible testbed for webpage development featuring the full use of html, css and javascript. The requirement was for something in between the defacto standard JSFiddle Code Playground and a full blown IDE.

To make the testbed more versatile the full Head section of the html code was included in the editor, affording the user full control from within the page.

JS-Quick compiles to a single file with inline css and javascript.

Test the JS-Quick webapp https://winderresource.org/jsquick


Future development could well see it expanded to include individual style and script files. This will depend on how well these features maybe received, after consulting user experience and feedback.

At the present time on-line files can always be included in the head section by using their complete url, which is particularly useful if you wish to include a responsive framework such as Bulma.
