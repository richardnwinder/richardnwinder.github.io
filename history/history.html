History

From the early days working with IBM 370 series, and the then mainstream cobol language.

Here is the COBOL source code for the rather famous Hello World example.

---- hello.cob ------------------------- * Hello World! COBOL program IDENTIFICATION DIVISION. PROGRAM-ID. hello. PROCEDURE DIVISION. DISPLAY "Hello World!". STOP RUN. ----------------------------------------

Checkout GnuCobol - versions to produce linux, windows and macOS executables from cobol source code - on sourceforge.io

....

During the 80's and 90's with the advances in C and C++, development was centred on CAD/CAM and CNC control, specialising in laser and rule processor development for the packaging industry.


....

Since 2010 most work has been on web-centric projects, and new languages such as Golang, NodeJs, Javascript and Python.
